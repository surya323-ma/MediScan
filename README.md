"# MediScan" 
# 🌿 Mediscan – AI-Powered Disease Prediction Platform

# 🚀 A smart healthcare platform that predicts diseases based on symptoms using Machine Learning & provides doctor recommendations.

# 📌 Overview

Mediscan is an AI-driven web application that helps users:

🧠 Predict possible diseases based on symptoms
👨‍⚕️ Get recommended doctors
📊 View health insights
🌐 Access everything through a simple web interface
✨ Features
🔍 Disease Prediction (ML आधारित)
👨‍⚕️ Doctor Recommendation System
📈 User-friendly Dashboard
🌐 REST API Integration (Django Backend)
🎯 Accurate & Fast Results

# 🛠️ Tech Stack
Category	Technology Used
💻 Frontend	React.js / HTML / CSS
⚙️ Backend	Django / Django REST Framework
🧠 ML Model	Python (Scikit-learn / Pandas)
🗄️ Database	SQLite / PostgreSQL
☁️ Deployment	Render

# ⚙️ Installation & Setup
🔹 1. Clone Repository
git clone https://github.com/surya323-ma/mediscan.git
cd mediscan
🔹 2. Backend Setup (Django)
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
🔹 3. Frontend Setup
cd frontend
npm install
npm start
#🌐 Deployment
Backend deployed on Render 
Make sure to:
Set ALLOWED_HOSTS
Add environment variables
Use gunicorn for production
